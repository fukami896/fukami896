同人場心得

這次令我感到最不一樣的，大概是拍照標準的放寬吧
Coser在待命區階梯下擺出pose讓攝影師們拍照，只要鏡頭沒有對著場內
大概都不會衝過去勸導他們一下
雖然相對於以前的嚴格標準來說，這樣究竟是好還是壞我也不知道
但是至少就我看到的來說，我還蠻喜歡這副景象的
比起以前階梯下方空蕩蕩的，這次這樣更有慶典的氣氛

以前因為請南大來協辦而認識的同屆的少佐
這次第一天也有遇到他來幫忙，這真的讓我很開心，也相當感動
但是南大這次因為工作分配的關係，沒甚麼辦法和他們有些交流，讓我覺得稍稍可惜

我還記得，剛進漫畫社的時候，就被刻難抓去當同人場的總務
當時因為自己的不認真與疏忽，出了很多包
還好有社團大家的幫忙，總算是讓活動順利結束
雖然事後檢討會不免被表了一頓XD”

當時第一次跟著張囧出外場，才知道原來廁所垃圾有這麼恐怖
邊收垃圾邊罵幹，雖然這景象想起來相當滑稽
但是，我想這是許多跑過外場的人的共同經驗XDD

然後是站垃圾桶，有了那兩天的經驗
即使活動過了幾個禮拜，我喝完飲料還是會不自覺的幫它分類一下www

最後，是兩天活動最後都一定會有的神秘儀式
看到麵條就這樣噴出來卡在那邊，真的是讓我笑到不行

隨著時間過去，出去帶外場、舉行儀式的人換成我了，現在的體力也大不如前
都不免讓我感到有點感傷^^”
不過，活動就是這樣一屆屆傳下去的
希望對每個人來說，這次的活動也會成為他們心中的一個美好回憶

我也要在這邊順便感謝所有參與這個活動的人
工作不分輕重，每個人都同樣重要
沒有你們的鼎力相助，成大同人場這個活動是絕對辦不起來的
謝謝！

—2012/05/16 阿諾

同場加映：
萬用模糊技法(Pixiv)，簡述如下：

  1. 弄個圖層0拷貝，放在0上面，模式為色彩增值
  2. Ctrl+A全選畫面，編輯>拷貝合併&貼上，模式為濾色，隨便高斯模糊一下
  3. 再弄個0拷貝，放在最上面，模式為變亮 —> 完成

真的隨便都超有質感，有沒有這麼爽www

Ubuntu筆記

系統：Ubuntu 14.04

  1. 在Ubuntu 做BottonMapping
    修改檔案：/usr/share/X11/xorg.conf.d/50-vmmouse.conf
    因為會是唯讀 所以要用Terminal輸入：sudo gedit來編輯
    打開之後複製下面的內容取代原本的 (我用的是Kensington Expert Mouse)
    最後存檔重開機 完成!

     Section "InputClass"
     Identifier  "Expert Mouse"
     MatchProduct    "Kensington      Kensington Expert Mouse"
     MatchIsPointer  "on"
     MatchDevicePath "/dev/input/event*"
     Driver      "evdev"
     Option      "SendCoreEvents" "true"
     #physical butons come from the mouse as:
     #   2 8
     #   1 3
     #This makes the top left and top right exchange
     #
     Option "ButtonMapping"  "1 8 3 4 5 6 7 2"
 EndSection

  2. 地區支援不見

     sudo apt-get install gnome-language-selector

  3. 輸入法有問題

     killall ibus-daemon
 sudo apt-get install ibus-daemon

  4. 預設的HDMI輸出音效沒有聲音，Google一下就有解決辦法了
